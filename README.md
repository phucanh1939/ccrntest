# CCRN Test

Test Cocos Creator game importer for React Native npm package [ccrn](https://github.com/phucanh1939/ccrn.git)

# How to run

Install ccrn 
```console
$ npm install ccrn -g
```

Install npm packages
```console
$ npm install
```

Import cocos engine
```console
$ ccrn init
```

Run server
```console
$ cd server
$ npm start
```

Start react-native
```console
$ npx react-native start
```

Build & run android
```
$ npx react-native run-android
```
